TSA Assistant — Prototype

Bienvenue — cet outil accompagne des personnes TSA/TDAH pour l'organisation, la planification et la gestion de la surcharge.

  • Démarrer l'onboarding (créer/éditer ton profil)
  • Page d'onboarding (direct)
  • Ouvrir le chat
  • Voir le schéma du questionnaire (JSON)
  • Voir le profil actuel (JSON)

API utiles :

  • GET /profile — récupérer le profil
  • PUT /profile — remplacer le profil
  • POST /ask — envoyer une question (body: { message, mode })
  • POST /onboarding/submit — soumettre le questionnaire

Remarque : c'est un prototype local. Les données sont stockées dans data/profile.json.
